## Struktur File

Project ini memiliki dua versi penyajian:

### 1. Versi PHP (Untuk Asisten Lab)
- **File:** `polos.php` dan `style.css`
- **Deskripsi:** Versi ini menggunakan pemrosesan PHP murni untuk logika penilaian. File ini khusus ditujukan untuk pemeriksaan oleh Asisten Lab.

### 2. Versi GitHub Pages (Publik)
- **File:** `index.html`, `static-logic.js`, dan `style.css`
- **Deskripsi:** Versi ini dioptimalkan untuk hosting statis di GitHub Pages. Logika PHP telah dikonversi menjadi JavaScript (`static-logic.js`) agar dapat berjalan tanpa server backend.
